# Katalyst Dashboard

A modern web dashboard interface for managing test cases, test runs, and key metrics.

This project includes:
- Sidebar navigation
- Dashboard statistics cards
- Test Runs panel
- Date range filter using Flatpickr
- Key Metrics panel
- Pass/Fail overview section
- Recent Activity section
